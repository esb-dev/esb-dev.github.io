Praktikum

Die Vorlesung wird kompakt in der ersten Hälfte des Semesters die Konzepte von Architektur und Implementierung von Datenbankmanagementsystemen vorstellen.

Das Praktikum in der zweiten Hälfte des Semesters besteht darin, eine Analyse der Architektur und Implementierung von H2 (eines Open Source DBMS, das in Java geschrieben ist) durchzuführen (Siehe Webseite der H2 Database Engine ⇗).
