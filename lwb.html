LWB Logic Workbench

The Logic Workbench (LWB) is a set of tools for playing with logic. It is written in Clojure and comprises tools for the propositional logic, the predicate (first order) logic and the linear temporal logic. LWB has a component for natural deduction in the mentioned logics.

LWB is a playground for logic and programming logic in Clojure.

More about LWB: Documentation ➚ and Source Code ➚ and Library on clojars ➚ .

It's quite convenient to use the logic workbench as a plugin in atom, see lwb-ui ➚ .
