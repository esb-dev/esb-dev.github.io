3.2. Prod🔗

Prod (Produkt) ist eine Struktur für geordnete Paare (2-Tupel). Wir hätten also den Typ Point aus Kapitel 2 auch als Prod definieren können.

Die Struktur definiert das kartesische Produkt mit den Selektoren fst und snd:

structure Prod (α : Type) (β : Type) : Type where fst : α snd : β

Für Prod gibt es eine spezielle Notation:

  • Statt Prod α β kann man α × β schreiben. (Man gibt × ein, indem man backslash gefolgt von x tippt.)

  • Man kann dann Werte des Typs angeben, indem man die Komponenten mit Komma getrennt in Klammern schreibt.

Mit Prod kann man auch n-Tupel erstellen: α × β × γ ist rechts-assoziativ, also α × (β × γ).

def point2d : Nat × Nat := (42, 42) (42, 42)#eval point2d 42#eval point2d.fst def point3d : Nat × Nat × Nat := (41, 42, 43) (41, 42, 43)#eval point3d 41#eval point3d.fst (42, 43)#eval point3d.snd true#eval point3d.snd.fst == 42 def sevens1 : String × Int × Nat := ("VII", 7, 4 + 3) def sevens2 : String × (Int × Nat) := ("VII", (7, 4 + 3)) true#eval sevens1 == sevens2
